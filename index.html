All Weather Portfolio Tracker

This application helps you track and manage your All Weather Portfolio investments using Google Sheets integration.

Features

  • Automated ETF price tracking
  • Portfolio performance monitoring
  • Monthly rebalancing calculations
  • Risk-adjusted weight calculations

Data Usage

This application requires access to:

  • Google Sheets API to store and manage your portfolio data
  • Read/write permissions to create and update spreadsheets
